# PingPongGame

A simple JavaScript-based browser Pong game.

##  Project Structure

- `index.html` — game canvas and UI
- `style.css` — basic styling for the game layout
- `script.js` — game logic, input handling, and rendering

## ▶ How to Run

1. Open `PingPongGame/index.html` in your browser.
2. Use the arrow keys to move the paddle:
   - **Up**: ↑
   - **Down**: ↓

##  Notes

- This is a lightweight demo game meant for learning JavaScript and canvas.
- Feel free to enhance it (scorekeeping, sound effects, difficulty levels, etc.).
